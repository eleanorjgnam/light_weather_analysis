1 Helper Functions

The nightdater function created rounds early-morning datetimes back to the previous night, creating a night of date. For example, a bird tagged at 23:00 on July 15 and a bird tagged at 3:00 on July 16 would both be considered tagged on the night of July 15

nightdater <- function (colIn) {
  nightdate <- as.Date(trunc(colIn - ifelse(lubridate::hour(colIn) < 8, 86400, 0), by = "days"))
  return(nightdate)
}

2 Weather Data

Our weather data from 2025 was collected via Kestrel 5500 weather meters. We had one on Gull Island, one on Great Island, and one on Baccalieu Island. We had persistent problems with the wind-measuring propellers breaking or getting jammed, so our record for wind data in particular is incomplete. One goal is to compare the island weather data with publicly available weather data from St. John’s International Airport to assess whether airport data can replace missing island-level measurements.

2.1 Standardization of Units for Comparison

In order for proper comparison to occur some values will have to be converted to different units due to differences in standards established by the lab for the Kestrel 5500 and the St. John’s International Airport. From here on out, the standard is as follows.

Standardized Units of Measurements
Variable Unit
datetime YYYY-MM-DD hh:mm:ss
temperature °C
station_pressure kPa
barometric_pressure kPa
wind_speed km/h
wind_chill °C
relative_humidity %
dew_point °C

2.2 Kestrel Import Function

Read in the data so the date and time readings are datetime objects and in Newfoundland time.

The Gull Island and Great Island kestrels had long periods where their wind impellers were broken. We also typically turned the kestrels on before they were deployed and did not turn them off until several days after retrieval, so I will trim the data to remove the extra days, and replace the ‘0’ wind speed with NA where the impellers were broken.

read_kestrel_weather = function(file, island, start_date, end_date, wind_all_missing = FALSE, wind_missing_after = NULL) {
  kestrel_weather = read_csv(file, show_col_types = FALSE) %>%
    mutate(
      datetime = as_datetime(Time, format = "%m/%d/%Y %H:%M"),
      datetime = force_tz(datetime, tzone = "America/St_Johns")
    ) %>%
    filter(datetime >= ymd(start_date, tz = "America/St_Johns"),
           datetime <= ymd(end_date, tz = "America/St_Johns"))
  
  if(wind_all_missing) {
    kestrel_weather = kestrel_weather %>%
      mutate(`Wind Speed` = NA_real_)
  }
  if(!is.null(wind_missing_after)) {
    kestrel_weather = kestrel_weather %>%
      mutate(`Wind Speed` = if_else(datetime > ymd(wind_missing_after), NA_real_, `Wind Speed`))
  }
  
  kestrel_weather %>%
    transmute(
      datetime,
      temperature = Temp,
      station_pressure = `Station P.` / 10,
      barometric_pressure = Baro. / 10,
      wind_speed = `Wind Speed` * 3.6,
      wind_chill = `Wind Chill`,
      relative_humidity = `Rel. Hum.`,
      dew_point = `Dew Point`,
      weather_station_location = island
    )
}

3 Light Data

Our light meters are Unihedron SQM-LU-DL (Sky Quality Meter Lens USB DataLogger) light meters, made for measuring nighttime light pollution. They measure light in Magnitudes per Arcsecond Squared, which is an astronomical measurement; magnitudes relates to star brightness (the sun’s magnitude is -27, Sirius has a magnitude of -1, Polaris has a magnitude of 2). Arc-seconds relate to how the night sky is divided into minutes and seconds; a square arc-second is a one-second by one-second area of the sky. If you could divide the light of a star with a magnitude of 20 (very dim; Pluto’s largest moon has a magnitude of 16) evenly over one square arcsecond, the MPSAS reading would be 20. The scale is negative, with higher values being darker. A sky with an MSAS reading of 5 (the cutoff for our meters) would be around sunset. Our night skies on dark nights are typically around 21 MPSAS, and the darkest night skies on earth read around 25. Each increase of 1 MPSAS means 2.5 times less light is coming from the sky; a sky measured at 5 MPSAS is 100 times brighter than a sky measured at 10 MPSAS.

We had one light meter on Great Island (for Witless Bay) in 2025 and one on Baccalieu. Not much data trimming needed here; I’ve done some pre-processing of the data already, and the light meters have been much less finicky than the weather meters.

baccalieu_light <- read_csv("data/baccalieu_light_20251023.csv", show_col_types = FALSE) %>%

One thing that may prove useful is filtering the light data for before/after (etc) nautical or astronomical twilight. The data frames have a sun elevation variable; nautical night begins when the sun is 12 degrees below the horizon, and astronomical night begins when the sun is 18 degrees below the horizon.

3.1 Sunrise and Sunset times

For doing anything with light data, it can be useful to have a list of sunrise and sunset times. Our islands are not so far apart that these need to be calculated separately. The package “StreamMetabolism” has a very useful function for calculating sunrise and sunset as new columns using a date column. However, the function is slow and computationally heavy, so I usually make a list of sunrise and sunset times for our study period, then join that list to other dataframes as needed.

suntimes  = tibble(
  date = seq(ymd("2025-06-01"), ymd("2025-10-31"), by = "1 day"),
  lat = 47.58,
  lon = -52.71
) %>% 
  rowwise() %>% 
  mutate(sunrise.set(lat = lat, long = lon, date = date + lubridate::days(1), timezone = "America/St_Johns")) ## Adding one day is necessary to get sunset for the given date rather than the night before

No Barometric Pressure is recorded, only station pressure. i will include the column only for comparisons sake. the elevation is taken from the same website as the data was taken from.

because the airport data is taken every hour on the half hour, the time will be rounded to match the data collected from the kestrel 5500 stations.

master_weather_data = airport_weather_data %>%
  mutate(datetime = as_datetime(Date.Time..LST., format = "%Y-%m-%d %H:%M"),
         datetime = force_tz(datetime, tzone = "America/St_Johns"),
         datetime = round_date(datetime, "hour")) %>%
  transmute(
    datetime,
    temperature = Temp...C.,
    station_pressure = Stn.Press..kPa.,
    barometric_pressure = station_pressure * ((1 - (0.0065 * 140.50) / (temperature + 273.15 + (0.0065 * 140.50)))^-5.257),
    wind_speed = Wind.Spd..km.h.,
    wind_chill = Wind.Chill,
    relative_humidity = Rel.Hum....,
    dew_point = Dew.Point.Temp...C.,
    weather_station_location = "St. John's International Airport"
  ) %>%
  filter(datetime >= "2025-06-10" & datetime <= "2025-10-29") %>%
  bind_rows(baccalieu_weather, gull_weather, great_weather)

5 Creation of Daily Summaries

5.1 Daily Summary Function

make_daily_summary = function(data, variable) {
  data %>%
    mutate(date = as_date(datetime)) %>%
    group_by(date, weather_station_location) %>%
    summarise(
      value = mean({{ variable }}, na.rm = TRUE),
      .groups = "drop"
    ) %>%
    pivot_wider(names_from = weather_station_location,
                values_from = value)
}

6 Correlation Matrices

6.1 Functions for ggpairs plots

ggpairs_lower_function = function(data, mapping) {
  ggplot(data = data, mapping = mapping) +
    geom_point(alpha = 0.5) +
    geom_smooth(method = "lm", se = FALSE)
}

ggpairs_diagonal_function = function(data, mapping) {
  ggplot(data = data, mapping = mapping) +
    geom_density()
}

cleaned_airport_weather_data = airport_weather_data %>%
  mutate(datetime = as_datetime(Date.Time..LST., format = "%Y-%m-%d %H:%M"),
         datetime = force_tz(datetime, tzone = "America/St_Johns"),
         datetime = round_date(datetime, "hour")) %>%
  select(datetime,Weather, Visibility..km., Temp...C., Stn.Press..kPa., Rel.Hum...., Hmdx, Dew.Point.Temp...C., Wind.Spd..km.h., Wind.Dir..10s.deg., Wind.Chill) %>%
  rename(
    weather = Weather,
    visibility = Visibility..km.,
    temperature = Temp...C.,
    station_pressure = Stn.Press..kPa.,
    relative_humidity = Rel.Hum....,
    humidex = Hmdx,
    dew_point = Dew.Point.Temp...C.,
    wind_speed = Wind.Spd..km.h.,
    wind_direction = Wind.Dir..10s.deg.,
    wind_chill = Wind.Chill
  ) %>%
  mutate(barometric_pressure = (station_pressure)*((1-(0.0065*140.50)/(temperature+273.15+(0.0065*140.50)))^-5.257)) %>%
  relocate(barometric_pressure, .before = relative_humidity) %>%
  mutate(weather = tolower(weather))

14 Evaluation of St. John’s International Airport Data For Island Approximation

14.1 Evaluation Function

evaluate_airport_prediction = function(data, island_name) {
  model_data = data %>%
    select(
      date,
      island_value = all_of(island_name),
      airport_value = `St. John's International Airport`
    ) %>%
    drop_na()
  
  model = lm(island_value ~ airport_value, data = model_data)
  
  tibble(
    island = island_name,
    n_days = nrow(model_data),
    r = cor(model_data$island_value, model_data$airport_value),
    r_squared = summary(model)$r.squared,
    root_mean_sqaure_error = sqrt(mean(resid(model)^2)),
    mean_absolute_error = mean(abs(resid(model))),
    bias_airport_minus_island = mean(model_data$airport_value - model_data$island_value),
    intercept = coef(model)[1],
    slope = coef(model)[2]
  )
}

14.2 Function to Plot Airport Against Island Data

plot_airport_prediction = function(data, island_name, y_label) {
  data %>%
    select(
      date,
      island_value = all_of(island_name),
      airport_value = "St. John's International Airport",
    ) %>%
    drop_na() %>%
    ggplot(aes(x = airport_value, y = island_value)) +
    geom_point(aes(colour = date), alpha = .7) +
    geom_smooth(method = "lm", se = TRUE) +
    labs(
      x = "St. John's International Airport",
      y = y_label,
      color = "Date"
    )
}
